# 프로젝트 이름

JSP, Servlet을 이용한 컴퓨터공학과 게시판

## 프로젝트 설명

이 프로젝트는 백엔드 기초인 JSP, Servlet 공부를 위해 개발되었습니다. 이 프로젝트는 백엔드 기초를 다지기 위한 게시판을 구현하는 데 중점을 두고 있습니다. Spring 프레임워크, NodeJS 등을 쓰지 않은 것이 이 프로젝트의 주요 특징입니다.
